# Mono

A sleek monochrome design system built with ShadCN. Minimalist, precise, and technical.

## Key Characteristics

- **Monochrome palette** — black, white, subtle gray
- **Grid-based layout** — boxed sections, precise alignment
- **Thin strokes & outlines** — no heavy fills
- **Dense data visualization** — metrics, coordinates, logs
- **Technical typography** — condensed / monospace feel
- **System-like micro-interactions** — progress bars, scanning, targeting

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **React 19** — latest features
- **Tailwind CSS 4** — utility-first styling
- **ShadCN UI** — accessible component primitives
- **Radix UI** — headless UI primitives
- **HugeIcons** — icon library

## Getting Started

```bash
pnpm install
pnpm dev
```

## Adding Components

```bash
npx shadcn@latest add button
```

## Usage

```tsx
import { Button } from "@/components/ui/button"
```
